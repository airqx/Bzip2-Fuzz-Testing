# Bzip2 Fuzz Testing

This project focuses on applying fuzz testing techniques to the Bzip2 compression algorithm. The purpose of this project is to identify vulnerabilities and bugs in the Bzip2 implementation by generating unexpected or random inputs.

## Overview
- **Project Name**: Bzip2 Fuzz Testing
- **Course**: SWE326
- **Date**: 2026-04-21

## Objectives
- Develop a fuzz testing framework suitable for Bzip2.
- Analyze the robustness of the Bzip2 algorithm against malformed inputs.
- Document any findings, including crashes or unexpected behavior observed during testing.
